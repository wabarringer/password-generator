Password Generator

- What was your motivation?
I was motivated to learn how to build something from javaScript
- What problem does it solve?
For the user, it solves the issue of having to come up with a new password
- What did you learn?
I learned the usefulness of for loops, charAt, Math.floor(Math.random)
How to push a value to an array
How to troubleshoot and properly run code as I go

- [Usage](#usage)
Activate using address:
https://wabarringer.github.io/password-generator/
To generate a password click "Generate Password"
Input the length of password you'd like (between 8 and 128 characters)
Confirm if you'd like numbers, symbols, upper case or lower case letters
If all prompted instructions were followed your new password will be in the box.
